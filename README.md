# Glam Confession

An experimental visual and editorial web project by **Bart Salazar / Simple Minds Studios**.

## About

*Glam Confession* explores image, identity, performance, atmosphere, and digital presentation through a stylized browser-based experience. The repository functions as both a creative artifact and a record of the project’s visual-development process.

## Project Status

Active creative showcase. Individual assets, dependencies, demonstrations, and external services may change over time.

## Repository Contents

This repository contains the project’s web files, visual assets, and supporting materials. Public availability is intended for viewing, discussion, portfolio presentation, and evaluation.
